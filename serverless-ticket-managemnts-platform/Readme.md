## 🧠 About the Project  

This project simulates a **real-world support ticket system**.  
Here, users can create tickets, track them, and update their status.  

The main focus of this project is:  
- Understanding serverless architecture  
- Understanding backend system flow  
- Building a real cloud project  

👉 In simple terms:  
**This is a backend system used by companies for issue tracking**

---

## ⚙️ How It Works  

System flow:

```
User → API Gateway → Lambda → DynamoDB → CloudWatch
```

- API Gateway → handles incoming requests  
- Lambda → executes backend logic  
- DynamoDB → stores data  
- CloudWatch → handles logs and monitoring  

👉 Fully serverless system (no server management)
